# Icons

Place the following PNG icon files in this directory:

- `icon16.png`  — 16×16 px
- `icon48.png`  — 48×48 px
- `icon128.png` — 128×128 px

The icons should use the GitHub mark (Octocat) or a custom Copilot-themed icon.
Any solid-color PNG files at those exact sizes will work for local development.
